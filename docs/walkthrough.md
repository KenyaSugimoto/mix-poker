# ウォークスルー - Seven Card Stud Hi MVP

Seven Card Stud Hiの初期MVPが実装されました。このアプリケーションは、シンプルな「Dark Luxury」デザインを採用し、5人のCPU対戦相手とプレイできるシングルプレイヤーモードを備えています。

## 実装された機能
- **ゲームエンジン**: アンティ、ブリングイン、ベッティングストリート（3rd〜7th）、ショーダウンを含む完全なSeven Card Studのロジック。
- **UIコンポーネント**:
  - **テーブル**: 楕円形の配置を採用したレーストラックスタイルのレイアウト。
  - **座席**: アバター、ハンド（カードが重なった表示）、チップ、アクションバブルを表示。
  - **アクションパネル**: フォールド、チェック、コール、ベット、レイズ、ブリングイン、コンプリートに対応したコンテキスト依存のボタン。
  - **アニメーション**: シンプルな勝利アニメーションとアクションフィードバック。
- **状態管理**: Zustandストアによる完全なゲーム状態の管理。
- **CPU**: 対戦相手用の基本的な戦略ロジック。

## 実行方法

1. **依存関係のインストール**（まだの場合）:
   ```bash
   npm install
   ```

2. **開発サーバーの起動**:
   ```bash
   npm run dev
   ```
   ブラウザで [http://localhost:5173](http://localhost:5173) を開いてください。

## ロジック概要
- **エンジン**: `src/features/stud/StudGameEngine.ts` がゲームの進行を制御します。
- **ストア**: `src/features/game/store/gameStore.ts` が状態を保持します。
- **ルール**: `src/lib/poker/studRules.ts` がブリングイン判定などのStud固有のルールを処理します。

## 既知の制限事項 / 次のステップ
- **ショーダウン判定**: 現在は簡易的です。`handEvaluator.ts` での `pokersolver` との統合は準備できていますが、エンジンは最初のパスとして仮の勝者選択を使用している場合があります（要確認）。
- **ベッティングロジック**: フィックスドリミットの構造は実装されていますが、サイドポットはまだ完全にはサポートされていません（十分なチップがあることを前提としています）。
- **モバイル**: レスポンシブレイアウトは実装されていますが、非常に小さな画面では調整が必要な場合があります。
